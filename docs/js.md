# JavaScript

## Installation

JavaScript実行環境(Node.js)をインストールする.

```bash
$ sudo apt install nodejs
```

## Version

```bash
$ node --version
v22.2.0
```

## Example

````{tab} Code
`hello.js`:

```js
console.log("Hello, World!");
```
````

````{tab} Console
```bash
$ node hello.js
Hello, World!
```
````
