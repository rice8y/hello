# Lua

## Installation

```bash
sudo apt install lua5.3
```

## Version

```bash
$ lua -v
Lua 5.3.6  Copyright (C) 1994-2020 Lua.org, PUC-Rio
```

## Example

### Script Mode

```lua
print("Hello, World!")
```

```bash
$ lua hello.lua
Hello, World!
```

### Interactive Mode

```bash
$ lua
Lua 5.3.6  Copyright (C) 1994-2020 Lua.org, PUC-Rio
> print("Hello, World!")
Hello, World!
```
