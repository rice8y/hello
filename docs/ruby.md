# Ruby

## Installation

Rubyをインストールする.

```bash
$ sudo apt-get install ruby-full
```

## Version

```bash
$ ruby --version
ruby 3.0.2p107 (2021-07-07 revision 0db68f0233) [x86_64-linux-gnu]
```
