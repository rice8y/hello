# nya~

## Installation

nya~インタプリタをダウンロードする.

```bash
$ git clone https://github.com/MichDem/Nya.git
```

インタプリタをコンパイルしてJavaクラスファイルを生成する.

```bash
$ cd Nya/src/org/nya
$ javac Nya.java Interpreter.java
```
