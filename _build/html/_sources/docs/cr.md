# Crystal

## Installation

Crystalをインストールする.

```bash
$ curl -fsSL https://crystal-lang.org/install.sh | $ sudo bash
```

## Version

```bash
$ crystal --version
Crystal 1.14.0 [dacd97bcc] (2024-10-09)

LLVM: 18.1.6
Default target: x86_64-unknown-linux-gnu
```
