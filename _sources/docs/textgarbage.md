# TextGarbage

## Installation

TextGarbageインタプリタをダウンロードする.

```bash
$ wget https://bin.smwcentral.net/u/41488/TextGarbageInterpreter.jar
```

## Example

````{tab} Code
`hello.tg`:

```none
ocHoceocloclocooc,oc ocWocoocroclocdoc!oc

```
````

````{tab} Terminal
```bash
$ java -jar TextGarbageInterpreter.jar hello.tg
Hello, World!
```
````
