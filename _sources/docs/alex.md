# Alex

## Installation

Alexインタプリタをダウンロードする.

```bash
$ git clone https://github.com/SLNETAIGA/AlexEsoLang.git
```

## Example

````{tab} Code
`hello.alex`:

```none
Say H - Alex.
Say e - Alex.
Say l - Alex.
Say l - Alex.
Say o - Alex.
Say , - Alex.
Say Space - Alex.
Say W - Alex.
Say o - Alex.
Say r - Alex.
Say l - Alex.
Say d - Alex.
Say ! - Alex.
Say LineFeed - Alex.
```
````

````{tab} Terminal
```bash

```
````
