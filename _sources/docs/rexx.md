# REXX

## Installation

REXXインタプリタであるReginaをインストールする.

```bash
$ sudo apt install regina-rexx
```

## Version

```bash
$ regina -v
REXX-Regina_3.6(MT) 5.00 31 Dec 2011
```

## Example

````{tab} Code
`hello.rexx`:

```none
say "Hello, World!"
```
````

